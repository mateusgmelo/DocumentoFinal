Diferença entre as linguagens

HTML, CSS e JavaScript são tecnologias fundamentais para o desenvolvimento web, cada uma com sua própria função e papel. HTML é a espinha dorsal da web, utilizado para criar a estrutura e o conteúdo básico de uma página, definindo elementos como cabeçalhos, parágrafos e links. CSS é responsável pela apresentação visual das páginas web, permitindo o controle sobre layout, cores, fontes e outros estilos para tornar o conteúdo mais atraente e acessível. Por outro lado, JavaScript é uma linguagem de programação dinâmica que permite tornar as páginas web interativas e dinâmicas. Ele é usado para criar funcionalidades como animações, validações de formulários, efeitos visuais dinâmicos e interações em tempo real com o usuário. Enquanto HTML estrutura a página, CSS a estiliza e JavaScript a torna interativa, juntos, essas tecnologias formam a base para o desenvolvimento web moderno, permitindo a criação de experiências online ricas e envolventes.

Paragrafos e Quebras

Emojis

Aqui, irmos abodar, de que maneira podemos fazer, para adicionar emojis ou simbolos, se maneira mais pratica ao nosso site. Por exemplo 🚀. Nesse emoji, eu usei o seguinte codigo, '&#x' para indicar que o valor que viria depois de x, seria o codigo de um emoji, que no meu caso, foi o codigo 1F680.

Favicon

Se voce analisar bem, esse site tem um favicon, que é nada esse simbolo que aparece ao lado do nome do site, que eu escolhi que fosse um emoji, exatamente esse 🚀.
Para que ele ficasse exatamente onde ele esta, eu usei o seguinte codigo: 


            <link rel="shortcut icon" href="favicon.ico" type="image/x-icon">
Colocando ele na cabeca do documento, onde chamamos de 'HEAD'.
